How to Fix a Stuck Windows Update Loop

When your computer says it is entirely up to date, but structural files are missing, it indicates a corrupted update distribution cache directory. Follow these step-by-step commands to reset it.

Step 1: Open Terminal As Administrator

Type cmd in your taskbar search box, right-click Command Prompt, and select Run as administrator.

Step 2: Halt Active Update Background Services

Run these four commands one by one, pressing Enter after each command to completely pause update locks:

net stop wuauserv
net stop cryptSvc
net stop bits
net stop msiserver

Step 3: Rename Corrupted Cache Folders

Run these commands one by one after the last ones are complete to isolate and rename old folders. Windows will automatically generate clean versions during the next scan:

ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
ren C:\Windows\System32\catroot2 catroot2.old

Step 4: Restore Background Services

Turn the internal delivery networks back online by running these commands one by one:

net start wuauserv
net start cryptSvc
net start bits
net start msiserver

Step 5: Restart and Sync

Type exit, hit Enter, and restart your computer. Open your normal settings layout and hit Check for updates to start fresh.
